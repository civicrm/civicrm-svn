1. Overview

    Importing data will probably be one of the first things you do with Enterprise. Please set aside enough time so that you can carefully step through this process and review the results. We will display your progress through the steps on the left, and you may cancel the Import at anytime. You can also undo any completed import in case you change your mind or want to start again. The Import process entails:

    1. Selecting a comma or tab delimited file (CSV) and uploading it to Enterprise
    2. We will then display the first several records of the uploaded file to verfiy its integrity
    3. You must then match the imported fields with the fields that Enterprise uses
    4. Once the fields are matched, you can then review the data and how it will appear in Enterprise
    5. You will then start the import process
    6. When the Import is completed, you can review the database and even perform de-duplication if you like
