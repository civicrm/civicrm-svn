How To:

  • Select one of the Import or Export processes from the list on the left. We will then guide you through a step by step process to import or export your records.
  • All data imported or exported must be in a tab or comma delimited text file (CSV file)
  • The Import Gifts Only and Import Activities Only processes can be used if you know the AccountIDs of your existing records and include them in the dataset to be imported.
  • Any Import process can be "Undone" and revert you back to your previous dataset
