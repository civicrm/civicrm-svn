Access Keys:

Use the following key combinations for:

  • ALT+SHIFT+I - New Individual
  • ALT+SHIFT+O - New Organization
  • ALT+SHIFT+H - New Household
  • ALT+SHIFT+G - New Group
  • ALT+SHIFT+A - New Activity
  • ALT+SHIFT+E - Edit Contact(View Contact Summary Page)
  • ALT+SHIFT+S - Save Button
  • ALT+SHIFT+N - Adding a new record in each tab (Contributions, Events,..etc)
