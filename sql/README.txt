Explanations for some of the file in this directory:

civicrm_41.mysql  : schema definition for mysql v4.1 and greater
civicrm_40.mysql  : schema definition for mysql v4.0
civicrm_data.mysql: Required data for civicrm
civicrm_generated.mysql: CiviCRM sample data, used for testing. This data file INCLUDES civicrm_data.mysql

civicrm_update_vOLD_vNEW_41.mysql: db upgrade file from OLD version to NEW version for mysql v4.1 and greater
civicrm_update_vOLD_vNEW_40.mysql: db upgrade file from OLD version to NEW version for mysql v4.0
