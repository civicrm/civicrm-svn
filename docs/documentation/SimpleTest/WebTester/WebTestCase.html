Inherited From Class SimpleTestCase

SimpleTestCase::SimpleTestCase() - Sets up the test with no display.

SimpleTestCase::assertExpectation() - Runs an expectation directly, for extending the tests with new expectation classes.

SimpleTestCase::assertFalse() - Will be true on false and vice versa. False is the PHP definition of false, so that null, empty strings, zero and an empty array all count as false.

SimpleTestCase::assertTrue() - Called from within the test methods to register passes and failures.

SimpleTestCase::createInvoker() - Used to invoke the single tests.

SimpleTestCase::dump() - Sends a formatted dump of a variable to the test suite for those emergency debugging situations.

SimpleTestCase::error() - Formats a PHP error and dispatches it to the runner.

SimpleTestCase::fail() - Sends a fail event with a message.

SimpleTestCase::getAssertionLine() - Uses a stack trace to find the line of an assertion.

SimpleTestCase::getLabel() - Accessor for the test name for subclasses.

SimpleTestCase::getSize() - Accessor for the number of subtests.

SimpleTestCase::pass() - Sends a pass event with a message.

SimpleTestCase::run() - Uses reflection to run every method within itself starting with the string "test" unless a method is specified.

SimpleTestCase::sendMessage() - Dispatches a text message straight to the test suite. Useful for status bar displays.

SimpleTestCase::setUp() - Sets up unit test wide variables at the start of each test method. To be overridden in actual user test cases.

SimpleTestCase::signal() - Sends a user defined event to the test runner.

SimpleTestCase::swallowErrors() - Cancels any outstanding errors.

SimpleTestCase::tearDown() - Clears the data set in the setUp() method call.

SimpleTestCase::_createRunner() - Can modify the incoming reporter so as to run the tests differently. This version simply passes it straight through.
