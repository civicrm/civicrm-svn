Located in File: /packages/Log/daemon.php

The Log_daemon class is a concrete implementation of the Log:: abstract class which sends messages to syslog daemon on UNIX-like machines.

This class uses the syslog protocol: http://www.ietf.org/rfc/rfc3164.txt

Method log (line 155)

void log( string $message, [int $priority = null])

Overrides : Log::log() Abstract implementation of the log() method.

Sends $message to the currently open syslog connection. Calls open() if necessary. Also passes the message along to any Log_observer instances that are observing this Log.

Parameters

  • string $message: The textual message to be logged.
  • int $priority: (optional) The priority of the message. Valid values are: LOG_EMERG, LOG_ALERT, LOG_CRIT, LOG_ERR, LOG_WARNING, LOG_NOTICE, LOG_INFO, and LOG_DEBUG. The default is LOG_INFO.
